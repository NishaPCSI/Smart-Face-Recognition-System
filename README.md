📸 Smart Face Recognition System

A simple Python-based face recognition project that lets you index & recognize faces using a pre-built face index. This project helps you build a basic face recognition pipeline that can detect, index, and recognize faces from a webcam or video feed.

⚙️ This project uses Python and common computer vision libraries to perform face identification. It doesn’t include a GUI; all interactions are via the command line or terminal.

🧠 Features

✔️ Build a face index from a dataset of images
✔️ Recognize faces in real time using the created index
✔️ Simple and effective for basic face recognition tasks
✔️ Uses Python libraries for detection and matching

📁 Project Structure
Smart-Face-Recognition-System/
├── IMAGES/                 # Face images used for indexing
├── indexer.py              # Build face index (encodings)
├── app2.py                 # Recognize faces using the built index
├── faces.index             # Stored face index generated by indexer
├── names.pkl               # Names corresponding to indexed faces
├── dlib-19.22.99-...whl    # Dlib wheel for face processing

📦 Requirements

Install dependencies:

pip install -r requirements.txt


Typical dependencies include:

opencv-python

dlib

face_recognition

pickle

(Install dlib manually if pip install fails – many users install the .whl file included.)

🚀 How It Works

This project has two main steps:

1. Build Face Index

Run the indexer.py script to process all images in your dataset and save face embeddings:

python indexer.py


This generates two files:

faces.index — encoded embeddings of faces

names.pkl — list of names matching the embeddings

These files are used later for recognition.

2. Recognize Faces

After indexing, run the recognition script:

python app2.py


This will:
✔️ Capture video from your webcam
✔️ Detect faces in real time
✔️ Compare them against the stored face index
✔️ Output recognized face names

🛠️ How to Add More Faces

Put a clear face image into the IMAGES/ directory.

Name the file with the person’s name (e.g., john_doe.jpg).

Re-run:

python indexer.py


This updates the index with new faces.

💡 Notes

The system uses classic face recognition libraries (e.g., dlib or face_recognition) to detect and encode faces.

Performance depends on the quality of face images and lighting conditions.

This is more suitable for learning and small projects; for large-scale or production deployments, consider optimized models or frameworks.

📌 Tips

✔️ When adding new people, ensure one clear face per image.
✔️ Re-index any time you add or remove images.
✔️ For better accuracy, use multiple images per person.
